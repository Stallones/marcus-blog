## Agent skills

### Issue tracker

Issue 以本地 Markdown 文件形式存储在 `.scratch/issues/` 目录下。

### Triage labels

无（个人项目，不使用标签分类）。

### Domain docs

领域文档位于 `docs/agents/`，按子目录组织：

- `docs/agents/frontend/` — 前端通用架构
- `docs/agents/ui-components/` — UI 组件体系
- `docs/agents/state-management/` — 状态管理（Pinia）
- `docs/agents/api-integration/` — API 集成层
- `docs/agents/routing/` — 路由系统
- `docs/agents/styling/` — 样式系统（Tailwind + SCSS）
