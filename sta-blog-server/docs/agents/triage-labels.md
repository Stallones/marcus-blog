# 分类标签

技能使用五个标准分类角色。此文件将这些角色映射为本仓库问题追踪器中实际使用的标签字符串。

| 技能中的角色           | 实际标签             | 含义                 |
| ---------------------- | -------------------- | -------------------- |
| `needs-triage`         | `needs-triage`       | 需要维护者评估       |
| `needs-info`           | `needs-info`         | 等待报告人补充信息   |
| `ready-for-agent`      | `ready-for-agent`    | 已明确，AI 可直接处理 |
| `ready-for-human`      | `ready-for-human`    | 需要人工实现         |
| `wontfix`              | `wontfix`            | 不处理               |

当技能提到某个角色时（如"应用 AI 可处理的分类标签"），使用上表中对应的实际标签字符串。
